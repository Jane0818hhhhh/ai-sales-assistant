# AI销售获客系统

基于WorkBuddy能力构建的营销获客辅助工具，参赛作品 for AI黑客松 - 方向三【营销】

## 项目简介

AI销售获客系统是为个人销售和小团队设计的智能辅助工具，旨在帮助销售人员提升获客效率、触达质量和转化管理能力。

## 核心功能

### 1. 客户背景速查
- 输入客户公司/人名，自动汇总公开信息
- 提取近期动态和关键联系人
- 推测决策链和潜在痛点
- 提供触达建议

### 2. 触达内容生成
- 基于客户背景自动生成个性化触达内容
- 支持微信、邮件、电话三种触达方式
- 提供使用提示和最佳实践

### 3. 异议处理教练
- 输入客户异议，获取应对策略
- 提供结构化应对框架
- 生成建议话术和类似案例参考

### 4. 跟进节奏管理
- 根据客户状态和销售流程阶段
- 智能生成下一步跟进计划
- 提供行动时机、渠道和目标建议

### 5. 销售过程复盘
- 粘贴通话记录或邮件往来
- 自动分析客户意向和成交概率
- 识别遗漏点和改进建议

## 技术架构

- **后端**: Python + Flask
- **前端**: HTML5 + CSS3 + JavaScript + Bootstrap 5
- **AI能力**: 集成WorkBuddy API（当前为Demo模拟版本）
- **部署**: 支持本地部署和云端部署

## WorkBuddy能力体现

本系统充分展示了WorkBuddy在以下方面的能力：

1. **信息抓取与聚合** - 客户背景速查功能
2. **自然语言生成** - 触达内容生成功能
3. **对话分析与理解** - 销售过程复盘功能
4. **结构化输出** - 所有功能的结果展示
5. **知识库与案例匹配** - 异议处理教练功能

## 安装与运行

### 环境要求
- Python 3.8+
- pip

### 安装依赖
```bash
pip install flask
```

### 运行应用
```bash
cd ai-sales-assistant
python app.py
```

访问: http://localhost:8080

## 使用说明

1. **客户背景速查**: 在首页点击"客户速查"，输入公司名称，点击查询
2. **触达内容生成**: 选择触达方式，填写客户信息和产品，生成个性化内容
3. **异议处理教练**: 选择或输入客户异议，获取应对建议
4. **跟进节奏管理**: 选择客户当前阶段，生成跟进计划
5. **销售过程复盘**: 粘贴对话记录，获取复盘分析
